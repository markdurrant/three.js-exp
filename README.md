# three.js experiments

experiment in using three.js to visualize height data grid for cutting with paper plotter.

### getting started

Install [node](http://nodejs.org/) & [gulp](http://gulpjs.com/)

`brew install node`, `npm install -g gulp`

Install packages

`npm i`

Run Gulp

`gulp`
